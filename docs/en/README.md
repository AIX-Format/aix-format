# 📚 English Documentation - AIX Format

Welcome to the official AIX Format standard documentation in English.

## 🎯 Overview

AIX Format is an open-source standard for sovereign AI agents, providing:
- 🔐 Cryptographic identity (DIDs)
- 🔌 MCP protocol for integration
- 💰 Built-in micro-payments
- 🛡️ Advanced security and digital signatures
- 📊 Analytics and performance metrics
